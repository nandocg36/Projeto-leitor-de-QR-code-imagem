# Projeto-leitor-de-QR-code-imagem

PWA para ler códigos QR a partir de imagens, PDF ou captura de ecrã (quando o browser permitir).

## GitHub Pages (a partir de `main`)

1. No repositório: **Settings** → **Pages**.
2. Em **Build and deployment**, em **Source** escolha **Deploy from a branch**.
3. Branch **main**, pasta **/ (root)** → **Save**.
4. O site fica em `https://<utilizador>.github.io/Projeto-leitor-de-QR-code-imagem/` (o caminho coincide com o nome do repositório).

O ficheiro `.nojekyll` evita que o GitHub Pages trate o site como Jekyll e ignore ficheiros que começam por `_`.
